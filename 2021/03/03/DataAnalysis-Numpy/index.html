Use Python for Data Analysis-Numpy

Posted on 2021-03-03 Edited on 2021-03-09 In Computer Science

Intro

NumPy(Numerical Python) is a library for the Python programming language, adding support for large, multi-dimensional arrays and matrices, along with a large collection of high-level mathematical functions to operate on these arrays. In this article, I will briefly introduce how to use numpy.

Reminder: ALl the codes are run in Jupter

Why NumPy is so efficient?

  1. Internally stores data in contiguous block of memory, so easy to retrieve.
    Has alogrithm library written in C that can operate on this memory without any type-checking or overhead.
    NumPy arrays also use much less memory than other built-in Python sequences.
  2. NumPy operations perform Complex Computations without need for Python loops.

For example, you can find numpy is faster and uses less memory than just python from the following code.

Multiple 2 for each data structure

ndarry

It is a container for homogenous data, all its elements must be of same type.

Create ndarray

1.array - Will create an array out of a list. For list of lists, will create a higher-dimensional array.

Here, dtype=np.float64 is actually default, you can choose int32 or data types

2.empty - Creates array without initializing values. May return uninitialized garbage values.

may return garbage values, may not just 0

3.zeros - Creates array initialized with zeroes.

4.arange - Create array initialized with range of values.

Change dtype

astype will create a new ndarry

Operations and functions

The operations shown below are on each entries if the shapes of the array are the same.

+, -, *, /, comparison
sqrt and exp
max or min between two array
mean, sum

mean and sum can be also used to caclulation in columns or rows.

cumsum, cumprod
Notice on slicing

The change on slicing will also influence the original array.

This happens because numpy is used for data analysis, create a new array will be a waste of memory.
If you wanna a copy, please use arr_slice = arr[5:8].copy.
