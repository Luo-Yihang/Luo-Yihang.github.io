How to quickly subtitle your video? Use Arctime!

Posted on 2021-02-09 Edited on 2021-02-13 In Studio Related

Intro

These days, due to the some school club activities, I need to subtitle lots of videos. After finishing this work, I am going to share some awesome ways to subtitile the video. This tutorial will definitely help you save your time if you will or need to subtitile some videos!


Arctime

Using Premier to subtitle videos is really a waste of time. Today, I am going to tell you how to use the specialized software Arctime. ArcTime is a awesome subtitle creation software, which can run on Mac, Windows and Linux. You can quickly create and edit timelines, and you can also efficiently edit and translate text.

How to Download

Go to the link http://arctime.cn/download-arctime.html.

If you are in China, just download via 主下载线路 (main download line). If not, just download via 备选线路2（全球) (alternative line 2(Global)). And remember to download the version that matches your computer(Mac, Windows or Linux).

How to Use

  1. Double click “Arctime Pro.exe”;
  2. Import audio or video files: File->Import Audio or Video Files;
  3. If you already have the subtitle files, just import relative files via File->Import SRT Subtitles or File->Import ASS Subtitles;
  4. If you don’t have subtitle files, you need to create a textfile. This textfile needs to contain subtitles and and each line is one subtitle. Therefore, the format should be the as shown in the following figure.
  5. Click the Quick Drag Icon;
  6. Then you can subtitle the video in this way, and one time you will subtitle one line in the textfile that you have imported in step 4;

More functions

Also, Arctime provide other functions like Automatic Speech Recognition (ASR), automatic translation and so on. Some features are available for a fee.
